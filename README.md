# Ball Obstacles Game

Небольшая браузерная игра, в которой игрок управляет шариком и проводит его через препятствия с помощью мыши.

# Demo

Живая версия игры: https://landing-about-car.netlify.app/

## Функционал
- Управление шариком кликами мыши (нажатие по полю).
- Препятствия и границы игрового поля.
- Простая анимация движения шарика и препятствий.
- Таймер времени.

## Технологии
- HTML5 (структура страницы)
- CSS3 (оформление и расположение элементов)
- JavaScript (логика игры, обработка кликов мыши, движение шарика)

## Как запустить
1. Склонировать репозиторий или скачать архив.
2. Открыть файл `index.html` в браузере.

## Дополнительно

Проект разработан с использованием нейросети Perplexity в качестве помощника для генерации идей и подсказок по JavaScript-коду.
